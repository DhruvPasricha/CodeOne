## Description

***_Codeone_** is a web app built on top of React which includes an **_IDE_** supporting multiple languages for programming in addition with a highly customizable **_Web Editor_** supporting real time changes*

**Languages Supported -**

> IDE :

```
1. C++
2. Python
3. Ruby
4. Swift
5. Kotlin
6. Java
```

> Web Editor :

```
1. HTML
2. CSS
3. Javascript
```
